## Installation

LovelyButton is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LovelyButton'

or

pod 'LovelyButton' , :git => 'https://github.com/vancancode/LovelyButton.git'
```
